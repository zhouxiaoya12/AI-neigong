# 自注意力从头实现

> **预计时间：** ~20 分钟  

---

## 🧠 核心公式

```
Attention(Q,K,V) = softmax(Q·K^T / √d_k) · V
```

**Q（Query）：** "我在找什么"
**K（Key）：** "我有什么"  
**V（Value）：** "我的内容是什么"

每个 token 对每个其他 token 计算一个相关度分数 → softmax 归一化 → 用分数加权取 V。

**为什么除以 √d_k？** 点积随维度增大而增大 → softmax 趋向 one-hot → 梯度消失。除 √d_k 保持方差稳定。

---

## 🏆 关键点

1. **自注意力 = 每个词看所有其他词，取相关部分的加权和。**
2. **√d_k 缩放不是细节——没有它梯度消失。**
3. **复杂度 O(n²)。** 序列长度翻倍，计算量翻四倍。

---

> **自注意力是所有 LLM 的心脏。"The cat sat on the mat"——模型知道"cat"和"sat"相关，因为注意力把它们的分数算高了。**
