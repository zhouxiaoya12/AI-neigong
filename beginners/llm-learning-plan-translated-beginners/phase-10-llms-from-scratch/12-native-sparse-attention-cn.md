# 原生稀疏注意力（NSA）：只关注重要的 token

> **原文：** Native Sparse Attention  
> **预计时间：** ~15 分钟  

---

## 🧠 核心洞察

标准注意力是 O(n²)——每个 token 看所有其他 token。长上下文（128K+）时这太贵了。

**NSA 每个 token 只关注用三个策略选出的"关键 token"：**
1. **压缩注意力** — 把连续 token 压缩成小块，注意力在块级别
2. **选择注意力** — 动态选出跟当前 token 最相关的几个 token
3. **滑动窗口** — 局部上下文永远保留

**结果：** 训练速度提升 9 倍（64K 上下文），推理延迟降低 12 倍。

DeepSeek 把 NSA 用在 V3 系列里，处理百万级 token 上下文。

---

## 🏆 关键数字

- 精度损失 < 1%
