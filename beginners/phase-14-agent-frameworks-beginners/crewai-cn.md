# CrewAI：基于角色的 Agent 团队

> **原文：** CrewAI  
> **预计时间：** ~15 分钟  

---

## 🧠 四个原语

| 原语 | 是什么 |
|------|--------|
| **Agent** | 角色 + 目标 + 背景故事 + 工具 |
| **Task** | 要完成的具体任务 |
| **Crew** | Agent + Task 的组合 |
| **Process** | 顺序执行 or 层级（管理者路由） |

---

## 🔀 Crew vs Flow

```
Crew: Agent 自己决定怎么协作（自主）
Flow: 事件驱动、确定性步骤（生产级）
```

**CrewAI 官方建议：生产应用从 Flow 开始。**

---

## 🏆 关键点

1. **Crew = 角色扮演型协作。** 每个 Agent 有 goal 和 backstory。
2. **Flow = 生产级。** 确定性、可审计、可回滚。
3. **四种记忆：** 短期、长期、实体、用户——但跨重启持久化要自己接。

---

> **CrewAI 让你像搭团队一样搭 Agent。研究员、作者、编辑——各司其职。但要上生产？用 Flow，别让 LLM 替你路由。**
