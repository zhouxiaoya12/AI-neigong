# 多 Token 预测：一次预测好几个词

> **原文：** Multi-Token Prediction  
> **预计时间：** ~15 分钟  

---

## 🧠 核心洞察

标准 LLM 一次预测下一个 token。MTP 一次预测后面 2-4 个 token。

```
标准: "The" → 预测 "cat" → 预测 "sat" → 预测 "on"
MTP:  "The" → 同时预测 "cat", "sat", "on"
```

**训练时多了额外的监督信号 → 收敛更快。推理时可以配合推测性解码 → 更快。**

DeepSeek-V3 在训练时用了 MTP 目标，提升了数据效率和下游性能。EAGLE-3 的草稿模型本质上是 MTP 的特化版本。

---

## 🏆 关键数字

- 训练效率提升 2-3x
- 推理时可作为"免费"的推测性解码草稿

---

> **一次预测多个 token = 训练更快、推理也能更快。DeepSeek 和 Meta 都在用。**
