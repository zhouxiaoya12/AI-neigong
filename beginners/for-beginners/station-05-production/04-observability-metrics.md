# 当 Agent 卡住了——可观测性与推理指标

> 预计阅读：20分钟 | 难度：进阶 | 前置条件：理解了 Agent 循环和生产部署
>
> 你将学到：Agent 慢了你该看什么？TTFT、TPOT、E2E 分别告诉你什么？OTel GenAI 怎么端到端追踪一次 Agent 运行？

---

## 🚗 开场翻车：Agent 有时 30 秒响应，有时 3 秒

用户投诉："我的 Agent 有时候飞快，有时候卡半天。不知道是 Agent 的问题还是模型的问题还是工具的问题。"

开发团队翻日志——日志只记录了 LLM 调用。看不到工具执行时间、看不到 MCP 服务器往返、看不到子 Agent 调度。**三个小时猜谜，最后发现是一个 MCP 服务器偶尔在冷启动时卡住。**

没有追踪 = 盲人摸象。

---

## 📊 四个必看指标

| 指标 | 全称 | 测量什么 | 好的值 |
|------|------|---------|--------|
| **TTFT** | Time To First Token | 用户点发送 → 看到第一个字 | P99 < 500ms |
| **TPOT** | Time Per Output Token | 首个 token 之后每个 token 的时间 | P99 < 25ms |
| **E2E** | End-to-End | 用户点发送 → 收到完整回复 | P99 < 3s |
| **有效吞吐** | Goodput | 满足所有 SLO 的请求占比 | > 99% |

**TTFT 告诉你是谁在堵门**——预填充慢还是排队久。
**TPOT 告诉你模型吐字快不快**——受批次大小和显存带宽影响。
**E2E 是用户体验**——TTFT + TPOT × 输出长度。
**有效吞吐是唯一的综合指标**——高吞吐低有效吞吐 = 你处理了很多 Token 但用户全跑了。

---

## ⚠️ 永远别看平均值

LLM 延迟是**右偏分布**。一批请求里：
- 一半 TPOT 约 7ms
- 20 个 TPOT 约 60ms
- **平均值：9ms** ← 看起来很美

**P99：65ms** ← 这就是用户每天遇到的实际延迟。

**始终报告 P50、P90、P99。平均值是骗人的。**

---

## 🔍 OTel GenAI——端到端追踪

OpenTelemetry GenAI 是 2026 年的追踪标准。一次 Agent 运行的追踪长这样：

```
Agent 调用 (invoke_agent)
 ├── LLM 调用 (chat)          ← 用了多少 token？多快？
 ├── 工具执行 (tool.execute)   
 │    └── MCP 调用 (mcp.call) ← 哪个 MCP 服务器慢了？
 ├── LLM 调用 (chat)
 └── 子 Agent (subagent.invoke)
```

**整个追踪在一个 trace ID 下，span 之间父子嵌套。** Datadog、Langfuse、Arize Phoenix、AgentOps 都原生支持。

### 关键属性

每次追踪记录：
- `gen_ai.request.model` — 用的哪个模型
- `gen_ai.usage.input_tokens` / `output_tokens` — Token 消耗
- `gen_ai.tool.name` — 调了什么工具
- `gen_ai.agent.name` — 哪个 Agent

## 🏆 焊死在脑子里的东西

1. **TTFT、TPOT、E2E 三个指标各看一个维度。** 只测一个等于闭着一只眼开车。

2. **平均值是骗子。看 P99。** LLM 延迟右偏分布，P99 才是用户感受到的。

3. **有效吞吐是唯一综合指标。** 高吞吐低有效吞吐 = 你跑得快但用户全走了。

4. **OTel GenAI = 一次插桩，所有后端都能用。** 追踪 Agent、LLM 调用、工具调用、子 Agent 全部嵌套在一个 trace 里。

---

## 📝 练习答案

**问题：第二次 LLM 调用花了 3000ms——正常应该在 500ms 以内。**

可能原因：
1. 上下文窗口太大（前一轮工具返回了大量数据）→ 预填充超时
2. 模型被其他请求挤占了（批次调度问题）
3. Token 限制太紧导致多次重试

**修：** 检查是预填充慢还是排队久 → 如果是上下文太长，压缩工具返回的结果。如果是排队，加 GPU 或优化批次策略。

---

> **Agent 慢不可怕——可怕的是你不知道为什么慢。TTFT 告诉你谁在排队、TPOT 告诉你模型吐多快、OTel Trace 告诉你哪一步卡住了。装上这三样，盲人摸象变成看 X 光片。**
