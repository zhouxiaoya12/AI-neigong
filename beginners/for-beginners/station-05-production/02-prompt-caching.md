# Prompt Caching：让你的 token 账单打一折

> 预计阅读：20分钟 | 难度：进阶 | 前置条件：理解了 Agent 推理过程
>
> 你将学到：两层缓存怎么让你的 API 费用降低 90%、为什么很多人"开了缓存账单反而涨了"、以及怎么正确使用。

---

## 🚗 开场翻车：开了缓存，账单翻了 5 倍

一个团队给他们的 Agent 启用了 Anthropic 的 Prompt Caching。想着"能省 90% 呢"。

月底看账单——**翻了 5 倍。**

为什么？他们的 Agent 对每个用户请求并行发出 10 个工具调用。10 个请求都有相同的系统提示。但 10 个请求几乎同时到达——第一个请求还在写缓存，后面 9 个都没命中。

**10 次缓存写入（每次是正常价格的 1.25 倍），0 次缓存读取（便宜 90%）。** 开了"省钱"功能，花得比以前还多。

---

## 🤔 两层缓存：L1 和 L2

### L2：提供商缓存——省在 API 端

你的 Agent 每次调 API 都有一个"系统提示"——它定义了 Agent 是谁、会什么、规则是什么。这部分基本不变。

**缓存做的事：** 提供商记住你系统提示的计算结果（叫 KV Cache，注意力机制里的 key-value 对）。下次用同样的提示 → 直接复用 → 只花 10% 的钱。

| 提供商 | 怎么开 | 缓存的输入价格 | 正常输入价格 |
|--------|--------|-------------|------------|
| Anthropic | 在请求里标注 `cache_control` | $0.30/百万 token | $3.00/百万 token |
| OpenAI | ≥1024 token 的提示自动缓存 | 约正常价的 10% | 正常价 |

**Anthropic 的关键细节：**
- 缓存 5 分钟 TTL：写入溢价 1.25 倍
- 缓存 1 小时 TTL：写入溢价 2 倍
- 读取：约正常价的 10%

### L1：语义缓存——直接跳过 LLM

在调 API 之前，先看"这个问题我答过类似的吗？"

把问题转成向量（embedding），在缓存里找余弦相似度 > 0.95 的历史记录。命中了 → 直接返回缓存答案，不调 API。

**命中率（来自社区数据，不是官方保证）：**
- 开放式聊天：10-15%
- 结构化 FAQ：40-70%
- 代码问题：20-30%
- 语音 Agent 重复提示：50-80%

## 💀 两个让缓存归零的反模式

### 反模式 1：并行请求

```
Agent 同时发 10 个 API 请求
   ↓
请求 1 在写缓存（需要 ~300ms）
   ↓
请求 2-10 在缓存写入完成之前到达 → 全部 MISS
   ↓
10 次写入（贵），0 次读取（便宜）
账单 × 5
```

**修：** 先发请求 1，等 300ms（缓存写入完成），再发请求 2-10。

### 反模式 2：动态内容污染

```
系统提示：
"你是助手。当前时间 14:32:17。用户 ID: abc123。今天是 2026-06-14..."
```

**每个请求的系统提示都不一样。** 每个请求都是"新提示"。零命中。

**修：** 把真正的静态部分标为可缓存，动态部分放外面的单独字段。

```
[缓存块] 你是助手。规则：1... 2... 3... [/缓存块]
[动态块] 当前时间：14:32:17。用户 ID：abc123。
```

一个真实案例：ProjectDiscovery 团队通过这种方式把命中率从 **7% 提升到了 74%**。

## 🏆 焊死在脑子里的东西

1. **缓存不是开关——是协议。** 开了不等于省钱。用错了等于多花钱。

2. **系统提示里分清楚"静态"和"动态"。** 静态放缓存块，动态放外面。

3. **并行请求是缓存杀手。** 先发一个等缓存写好了再发其他的。

4. **L1（语义缓存）+ L2（提供商缓存）= 最大省钱。** 语义缓存直接跳过 API 调用，提供商缓存让必须调的便宜 90%。

---

## 📝 练习答案

```
[缓存块]
你是代码审查员。规则：1.检查安全性 2.检查性能 3.检查可读性。
[/缓存块]
[动态块]
当前项目：my-app。当前分支：feature/login。当前时间：14:32:17。
[/动态块]
```

**判断标准：** 如果一段文本在 5 分钟/1 小时内不会变 → 放进缓存块。跟用户/会话/时间相关的 → 放动态块。

---

> **缓存是给账单打的疫苗——用对了省钱，用错了反而发烧。静态进缓存、动态放外面、并行先排队——就这三条，做到了你的 token 账单能打一折。**
